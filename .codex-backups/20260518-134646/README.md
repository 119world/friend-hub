# Friend Hub

Low-budget Firebase + React MVP for login, profiles, discovery, matches, realtime chat, voice-call billing button, Razorpay recharge, and admin controls.

## Apps

- `frontend`: user mobile-first web app
- `admin`: admin dashboard
- `backend`: Express API with Firebase Admin and Razorpay
- `firebase`: Firestore and Storage rules

## Quick Start

```bash
npm run install:all
cp frontend/.env.example frontend/.env
cp admin/.env.example admin/.env
cp backend/.env.example backend/.env
npm run dev
```

Frontend: http://localhost:5173  
Admin: http://localhost:5174  
Backend: http://localhost:8080
